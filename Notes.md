# Planteamiento

App/Web App/Proyecto que actue como una librería para los libros digitales, como si fuese una física, pero digital.

Funcionalidades:

- Registro/Inicio de sesión || En el caso de hacerlo como una app de escritorio esto no sería necesario
- Conectar a Kindle por usb y que se carguen todos los libros (idea inicial se puede retocar)
- Poder añadir libros manualmente conectando la API de GoodReads para poder tenerlos todos
- Poder valorar las lecturas
- Poder cambiar títulos, imagenes y demás características de cada libro añadido
- Tener diferentes fondos, estanterías, temáticas donde poder alojar los libros y verlos
- Poder seleccionar un libro y ver sus características
- Otras
